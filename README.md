# opentrack-api-examples

A collection of scripts allowing you to interact with Opentrack.
The only requirement is Python's `requests` library

Please read https://docs.opentrack.run/appi/

In general, you'll need meeting director access to yoour own competitions
and to adapt the competition IDs.


This repo is being rewritten and updated in June 2026.  Please ignore the folder of old scripts,
as we've changed our API engine.  We hope to have them redone shortly.
